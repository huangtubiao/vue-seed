# 兼容说明

**iOS 兼容**

- iPhone5, iPhone5S, iPhone5C
- iPhone6, iPhone6S，iPhone7
- iPhone7S，

**Android 兼容四大产品**

- 小米
- 华为
- 魅族
- 三星

对于微信来说，iOS 端都是使用 safari，基本没有什么兼容性的问题，
而 Android 端，微信内置了其自行开发的 X5 内核，有一些兼容性问题，
但两者都是 webkit 内核，所以不需要兼容非 webkit 内核的浏览器。
