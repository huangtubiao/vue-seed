# qianfan-app

> A Vue.js project

## 技术栈

- Vue2
- vue-router2
- Vuex
- webpack2
- ES6 + Babel
- Sass/Less
- Eslint
- Koa

## Dependences

- node^7.2.0
- npm^4.0.5

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
