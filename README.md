# qianfan-app

> A Vue.js project

## 技术栈

- Vue2
- vue-router2
- Vuex
- webpack2
- ES6 + Babel
- Less
- Eslint

## Dependences

- node^7.2.0
- npm^4.0.5

## Build Setup

``` bash
# 安装依赖
npm install

# 启动服务，打开链接 localhost:9001
npm run dev

# 发布代码
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

> 在dev环境下，需要启动mock server来加载数据，进入"./mock"目录，按照README的命令步骤来启动服务。

重点：在开发之前，请认真阅读项目开发规范（./docs/）。
